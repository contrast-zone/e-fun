3.1. making a local copy

F

irst, we have to make sure we have installed a local web server like Apache HTTP Server project or similar software. We also have to make sure to have an access to a file manager and a text editor which we will use to access and edit E-Fun files.

To install E-Fun, we have to download the E-fun software bundle from dedicated github pages After downloading the software, we have to unpack the downloaded zip archive to a local server folder. Unpacking the archive will produce the following file-folder structure: 


└─┬ e-fun-home-folder
├─┬ contents
│ ├── contents.xml
│ └── ... contents go here ...
│
├─┬ media
│ └── ... internal ui files ...
│
├── changeslog.md
├── html2canvas.js
├── index.html
├── orbital.html
├── orbital.js 
├── orbital.json
├── README.md
└── LICENSE

Now that we have unpacked the archive, we can test it by entering the local HTTP server address into a web browser.
