hese pages are created in E-fun, and they contain E-fun use instructions. The current version of E-fun represent a progressive headless, flat-file, and light content management system. Content management system (CMS) is a software application that manage the creation and modification of digital content. Using E-fun, the content pages are supposed to be collected from outer resources such as image and HTML editors, while E-fun itself manages only tree structuring and browsing the contents. Supported content pages include HTML, XHTML, SVG, PNG, GIF, and JPEG files.
